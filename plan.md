# Дипломная работа: Python SDK с геопространственными возможностями для CDEK API

## 1. Описание проекта

### 1.1 Суть проекта

Разработка Python SDK для работы с API службы доставки CDEK с интегрированными геоинформационными возможностями. Проект решает задачу автоматизации работы с логистическим API и добавляет геопространственный анализ данных о пунктах выдачи, зонах доставки и расчете оптимальных вариантов доставки на основе геолокации.

**Основные компоненты:**
- Асинхронный Python SDK для полного покрытия CDEK API v2
- Геопространственный модуль для работы с координатами и расстояниями
- Веб-приложение для демонстрации возможностей (карта с офисами, калькулятор)
- Документация и примеры использования

### 1.2 Актуальность

**Проблемы:**
1. Отсутствие официального Python SDK для CDEK API v2
2. Сложность интеграции логистических API в приложения
3. Необходимость вручную искать ближайшие пункты выдачи
4. Отсутствие визуализации сети доставки на карте
5. Нет инструментов для анализа покрытия территории

**Востребованность:**
- 500+ тысяч интернет-магазинов в России нуждаются в интеграции с доставкой
- CDEK — один из крупнейших операторов (15 тысяч пунктов выдачи)
- Python — популярный язык для backend разработки и data science
- ГИС-технологии становятся стандартом в логистике

### 1.3 Цель работы

Разработать Python SDK для CDEK API с интегрированными геоинформационными возможностями для автоматизации работы с логистическим API и пространственного анализа данных о доставке.

### 1.4 Задачи

1. Проанализировать CDEK API v2 и определить требования к SDK
2. Разработать асинхронный SDK с полным покрытием API endpoints
3. Реализовать геопространственный модуль для работы с координатами
4. Создать веб-приложение для демонстрации возможностей
5. Написать тесты и документацию
6. Подготовить примеры использования для различных сценариев

- Поддержка test/production режимов

**Геопространственный модуль:**
- Геокодирование адресов (Yandex Geocoder)
- Расчет расстояний между точками (Haversine)
- Поиск N ближайших офисов от координат
- Фильтрация офисов по радиусу
- Интеграция с основными сервисами SDK

**Примеры использования:**
```python
from cdek import CdekClient

async with CdekClient(client_id, client_secret) as client:
    # Найти ближайшие офисы от адреса
    offices = await client.office.find_nearest(
        address="Москва, Красная площадь, 1",
        limit=5
    )
    
    # Расчет доставки с геокодированием
    result = await client.calculator.calculate_with_geocoding(
        from_address="Москва, ул. Ленина, 1",
        to_address="Санкт-Петербург, Невский проспект, 1",
        packages=[{"weight": 1000, "length": 10, "width": 10, "height": 10}]
    )
```
